# Registro de Empleados y Manejo de Archivos

## Descripción
Este proyecto consiste en una aplicación de escritorio que permite registrar datos de empleados. El objetivo es capturar la información, validar que no falte nada y guardarla en un archivo de texto de forma organizada.

El programa permite ingresar el ID, nombre, apellidos, dirección, teléfono, email, cargo, salario, fecha de ingreso y género. Incluye alertas para evitar campos vacíos y permite elegir dónde guardar el archivo usando un cuadro de diálogo profesional.

## Tecnologías Utilizadas
 **Lenguaje:** C#
 **Herramientas:** Windows Forms (.NET Framework)
 **Entorno o IDE:** Visual Studio 2026
 **Manejo de Datos:** Archivos de texto (.txt)

## Uso o Ejecución
1. Abrir el proyecto en Visual Studio.
2. Ejecutar la aplicación presionando F5.
3. Llenar los campos solicitados del empleado.
4. Hacer clic en "Guardar" y elegir la carpeta de destino.
5. El sistema abrirá el bloc de notas con la tabla lista.
